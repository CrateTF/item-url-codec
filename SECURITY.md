# Security policy

## Supported versions

Security fixes are provided for the latest released major version.

## Reporting a vulnerability

Do not disclose suspected vulnerabilities in a public issue. Use GitHub's
private vulnerability reporting feature for this repository when available, or
contact the CrateTF maintainers through the security contact published on
https://crate.tf/responsible-disclosure.

Include the affected version, a minimal reproduction, expected impact, and any
suggested mitigation. Do not include credentials, private production data, or
unrelated user information.

This package is intentionally dependency-free at runtime and performs no
network, credential, or filesystem operations.
