## [1.0.0] - 2026-07-21

### Added

- Canonical TF2 SKU normalization and Crate.tf item path generation.
- Grammar-aware parsing for canonical and supported legacy item path segments.
- Absolute Crate.tf item URL generation.
- Language-neutral contract documentation and portable conformance vectors.
- Strict TypeScript build, tests, package validation, and GitHub Actions CI.
